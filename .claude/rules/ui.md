# UI Rules (summary — full detail in docs/DESIGN_SYSTEM.md)

- Mobile-first. Write mobile styles first, then md: and lg: breakpoints.
- Use brand tokens only: brand-black, brand-dark, brand-surface, brand-border, brand-muted, brand-light, brand-white, brand-accent (#e8c97e gold), brand-red.
- No arbitrary hex values. No inline styles.
- font-display: Cormorant Garamond (headings, hero, blockquotes).
- font-body: Manrope (nav, buttons, body, admin).
- shadcn/ui components: copy-paste only. Never run `npx shadcn add`.
- Split files that exceed ~150 lines.
- Standard: Netflix-level browsing, Apple-level spacing, A24-level restraint.
- No card info panels below posters. No gold overuse.
